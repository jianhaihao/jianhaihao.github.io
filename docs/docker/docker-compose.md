# docker-compose

## docker-compose.yml 常用命令

|       命令        |        用途        |
|    ---------     |      ----------    |
|    build         |   本地创建镜像       |
|    command       |   覆盖缺省命令       |
|    depends_on    |   连接容器          |
|    ports         |   暴露端口          |
|    volumes       |   卷               |
|    image         |   pull镜像         |


## docker-compose 命令

|       命令        |        用途        |
|    ---------     |      ----------    |
|    up            |   启动服务          |
|    stop          |   停止服务          |
|    rm            |   删除服务中的各个容器|
|    logs          |   观察各个容器的日志  |
|    ps            |   列出服务相关的容器  |
