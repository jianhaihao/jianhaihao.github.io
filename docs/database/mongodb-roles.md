# mongodb 角色

**数据库用户角色**

`read`, `readwrite`

**数据库管理角色**

`dbAdmin`,`dbOwner`,`userAdmin`

**集群管理员角色**

`clusterAdmin`,`clusterManager`,`clusterMonitor`,`hostManager`

**备份和恢复角色**

`backup`,`restore`

**所有数据库角色**

在`admin`数据库可用

`readAnyDatebase`,`readWriteAnyDatabase`,`userAdminAnyDatabase`,`dbAdminAnyDatabase`

**超级管理员角色**

`root`
